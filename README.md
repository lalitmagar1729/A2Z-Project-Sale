# A2Z Project Sale

A full-stack e-commerce platform for selling programming projects and source code.
Built using Node.js, Express, EJS, and MySQL.

---

## 🚀 Features

* User authentication (Login / Register)
* Admin dashboard
* Product listing (projects & source code)
* Category-wise browsing
* Dynamic pages using EJS
* Error handling (404 / 500 pages)

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, Bootstrap, EJS
* Backend: Node.js, Express.js
* Database: MySQL (mysql2)
* Version Control: Git & GitHub

---

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/your-username/your-repo-name.git

2. Navigate into project

cd your-repo-name

3. Install dependencies

npm install

4. Create .env file

PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=your_database

5. Run the project

npm start

---

## 🔐 Environment Variables

Make sure to create a `.env` file and never push it to GitHub.

---

## ❗ Important Notes

* `node_modules` is ignored using `.gitignore`
* Do not upload `.env` file
* Use MySQL server running locally
